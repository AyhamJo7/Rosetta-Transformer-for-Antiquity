# Support

Thanks for your interest in **Rosetta Transformer for Antiquity** — a multilingual NLP system for ancient and low-resource languages.

## Getting help

- **Usage questions & ideas** — open a [GitHub Discussion](https://github.com/AyhamJo7/Rosetta-Transformer-for-Antiquity/discussions) (or an issue if Discussions is not enabled).
- **Bugs** — open a [GitHub Issue](https://github.com/AyhamJo7/Rosetta-Transformer-for-Antiquity/issues) using the bug template. Include your environment, a minimal reproduction, and the full traceback.
- **Feature requests** — open an issue with the feature template and describe the use case.

## Before opening an issue

1. Read the [README](README.md) — it covers setup, model configuration, and usage.
2. Search [existing issues](https://github.com/AyhamJo7/Rosetta-Transformer-for-Antiquity/issues?q=).
3. Confirm your Python version and that any required model weights or corpora are available locally.

## Security

Please do **not** report security vulnerabilities in public issues. See [SECURITY.md](SECURITY.md) for private disclosure.

## Response expectations

This is a research/portfolio project maintained on a best-effort basis — there is no commercial SLA.
